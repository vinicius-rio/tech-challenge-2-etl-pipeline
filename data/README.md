# Dados do projeto

Esta pasta documenta os dados utilizados no projeto.

Os arquivos de dados não são versionados no GitHub para evitar o
armazenamento de informações pesadas no repositório.

## Fonte

Base dos Dados — Avaliação da Alfabetização.

## Entidades utilizadas

- UF;
- Município;
- Dados de alunos;
- Meta de alfabetização do Brasil;
- Meta de alfabetização por UF;
- Meta de alfabetização por município.

## Armazenamento no Databricks

Os arquivos originais serão armazenados em um Volume do Unity Catalog e
posteriormente processados nas camadas Bronze, Silver e Gold.
